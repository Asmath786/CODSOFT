Task_4 - Student Management System
## Overview

The Student Management System is a console-based Java application developed to manage student records efficiently.
This project demonstrates the implementation of core Object-Oriented Programming (OOP) concepts such as classes, objects, encapsulation, and modular design.

The system allows users to add, remove, search, and display student details using a simple and interactive menu-driven interface.

## Features

-Add new student records
-Remove existing students
-Search student by roll number
-Display all student records
-Input validation for required fields
-Clean and structured code design

## Technologies Used

-Java
-OOP Principles
-ArrayList (Java Collections Framework)
-Scanner for user input

## Project Structure

Multi-file version (for GitHub):
-Student.java
-StudentManagementSystem.java
-Main.java

Single-file version (for Online Compiler):
-Main.java (All classes merged)

## How to Run

Using Command Prompt / VS Code
javac *.java
java Main

Using Online Compiler
-Copy the merged code.
-Paste into an online Java compiler.
-Click Run.
-Interact using the console menu.
