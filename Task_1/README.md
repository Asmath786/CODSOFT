# Task 1 – Number Guessing Game

## Description
This is a Java console-based number guessing game developed as part of the CodSoft Java Programming Internship.
The program generates a random number between 1 and 100 and allows the user to guess it within limited attempts. Feedback is given for each guess, and the user can play multiple rounds.

## Features
- Random number generation
- User input validation
- Limited attempts
- Option to play multiple rounds
- Score tracking

## How to Run
1. Compile the program:
   javac main.java
2. Run the program:
   java Main

## Technologies Used
- Java (JDK 17)
