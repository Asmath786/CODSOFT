# Task 4 – Currency Converter

##  Description
This project is a Java-based Currency Converter application developed as part of my internship tasks.  
It allows users to select a base currency and a target currency, enter an amount, and receive the converted value.


##  Features
- Currency selection (USD, INR, EUR, GBP, JPY)
- User amount input
- Currency conversion using predefined exchange rates
- Input validation
- Clear and formatted result display


##  Technologies Used
- Java
- Object-Oriented Programming (OOP)
- HashMap (for storing exchange rates)
- Scanner (for user input)


##  Project Structure
Task_4/
- CurrencyConverter.java
- Main.java


##  How to Run
1. Compile the program:
   javac Main.java CurrencyConverter.java

2. Run the program:
   java Main

- Class & Object interaction

- Data storage using HashMap
- User input handling
- Basic financial calculation logic
